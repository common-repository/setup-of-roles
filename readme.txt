"Setup of roles" very simple plug-in for setup of roles and capabilities of users.

== Description ==

By means of a plugin "Setup of roles" you can easily add/delete the user roles, except a role "Administrator", and quickly set up capabilities of each of them according to the needs.

== Installation ==

1. Extract "setup_of-roles.zip" archive content to the "/wp-content/plugins/" directory.
2. Activate the plugin through the 'Plugins' menu in WordPress. 
3. Change your WordPress standard roles capabilities according to your needs through the 'Options' menu in WordPress.

== Frequently Asked Questions ==

- Whether it is possible to rename a role?
No, so far such opportunity isn't present.

- Whether copying of roles is possible?
This opportunity will be provided in upcoming versions of a plugin.
